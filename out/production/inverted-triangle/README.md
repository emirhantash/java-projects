# **Inverted Triangle**

*We write a program that takes the number of digits from the user and draws an inverted triangle to the screen with asterisks ( * ) using loops in Java*

### *Example*

*Number of Digits: 10*

```
 *******************
  *****************
   ***************
    *************
     ***********
      *********
       *******
        *****
         ***
          *
```
